# math-js
Some Math function written in JS


### Square

##### Area
```javascript
SquareArea(s){
  let area = s*s;
  return area;
}
```

##### Perimeter
```javascript
SquarePerimeter(s){
  let perimeter = 4*s;
  return perimeter;
}
```

### Circle

##### Area
```javascript
CircleArea(r){
  let area = Math.PI*r*r;
  return area;
}
```

##### Perimeter
```javascript
SquarePerimeter(r){
  let perimeter = 2*Math.PI*r;
  return perimeter;
}
```
